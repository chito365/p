Key concepts in Information Security. Information Security in Networked Enterprises. Threats and vulnerabilities analysis. Effective System Administration. Policies. Risk management. ICT Security planning. Operational issues in ICT security (incident handling, training, backups etc). Physical security. Personnel issues. Types and uses of security devices. Network Security; (identification and authentication, logical access control, Routers, Proxies, and Firewalls audit trails and cryptography). Security for Electronic Commerce, Financial Networks, Intranets and Extranets. Security Across Different Operating Systems and Platforms. Detection of security breaches. Continuity and Disaster Recovery Planning.
